Poker:
This is a Five Card Draw Poker game. This one of the first poker games many learn whnen they enter the world of poker.
This program simulates that game using python Tkinter graphics and method. 
The user should run the fil poker.py to start the game.
Then he/she can click to either play the game or look at the rules and start the game
At the end of the game, the user can press r to restart or close the window to exit the game completely.
